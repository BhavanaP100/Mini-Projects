# 🎮 Battle of Hands

A simple Rock Paper Scissors game built using HTML, CSS and JavaScript.

## 🚀 Features

- Play against the computer
- Random computer moves
- Live score tracking
- Winner announcement
- Responsive UI (Work in Progress)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
